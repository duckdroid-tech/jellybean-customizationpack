# Readme 2 - Installing your Nova Launcher
## Installing the Nova Launcher APK version
Download Nova Launcher on your file manager, the file name of "nova-launcher-81006-8-1-6.apk". Click the app package file where you install, then, appears the "Stagging app" popup. After the downloading the APK, they appears that "Action not allowed" popup, while cannot install APK file. Tap "Learn More" button to see the device admin settings to turn off device admin apps.
## Installing the Nova Launcher on Google Play
Download and install the latest version of Nova Launcher on the Google Play.
